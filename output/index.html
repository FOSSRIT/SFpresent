Sash

  1. Activity for displaying the user's badges
  2. Reads datastore object for badge information
  3. Generates a badge from information received from DS object
  4. Displays badge(s) onto the window
  5. Tooltips provide useful information about the badges

How does Sash work?

  1. Activities that award badges use a badges library
  2. Badges lib creates a DS object with a specific property: has_badges
  3. Sash finds any DS object that has property 'has_badges: True'
  4. Sash checks if that activity has awarded the user any badges
  5. If so, Sash reads a symbolic link of the badge images that the badges lib created
  6. Badges are then displayed in Sash with tooltip information such as the name, criteria, data acquired, etc.

How to get translations to work

  1. $python setup.py genpot
  2. Generates a po/ directory with a <bundle_name>.pot file (rename to <bundle_name.po)
  3. Edit the .po file and add your translations
  4. $python setup.py dist_xo
  5. Generates the locale/ directory where it stores the new .mo file
  6. Add these next two lines of code at the beginning of the program
  7. import gettext
  8. lang = gettext.translation(<bundle_id>, 'locale/', languages=[<name_of_mo_file>])
  9. _ = lang.ugettext
  10. Every word you want translated, change it to _(<string>)
